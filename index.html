Core Capabilities

Built for Operators

Everything you need for efficient chain management and target tracking in one powerful desktop application.

Live Target Tracking

Real-time status updates with configurable refresh intervals. Track hundreds of targets simultaneously with instant visibility into who's attackable, hospitalized, traveling, or in jail.

AES-256 Encryption

Your API key is encrypted with AES-256-GCM before being stored. Sandboxed renderer with preload bridge ensures maximum security for your credentials.

Smart Rate Limiting

Built-in rate limiter with exponential backoff, jitter, and Retry-After support. Automatic handling of HTTP 429 ensures you never exceed Torn's API limits.

VS Code-Inspired UI

Familiar dark theme with activity bar and sidebar navigation. Keyboard-first workflow with shortcuts for every action. Compact mode for maximum density.

Target Groups

Organize targets into custom groups with color coding. Quick filters for attackable, hospital, jail, traveling, and favorites. Bulk import and export capabilities.

NPC Loot Timers

Track loot levels for all Torn NPCs and bosses. Visual progress bars show time until next level. Never miss maximum loot timing again.

Bounty Tracker

Manual bounty tracking with target IDs and reward amounts. Track your bounty statistics including collected, placed, and total earned.

Backup & Restore

Full backup system for targets, groups, attack history, statistics, and settings. Export to JSON and restore anytime. Never lose your data.
