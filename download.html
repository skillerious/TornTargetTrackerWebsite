Security

Verify your download

Always verify installer integrity before running. Match the hash from GitHub releases.

How to verify on Windows

Open PowerShell and run this command to calculate the file hash:

Get-FileHash TornTargetTrackerSetup-2.7.9.exe -Algorithm SHA256

Compare the output with the hash published on the GitHub release page.

Why verify?

Verification ensures the file hasn't been corrupted during download or tampered with. If hashes don't match, re-download from GitHub.

Support

Frequently asked questions

Common questions about downloading and installing Torn Target Tracker.

Is this safe to download?

Yes. Torn Target Tracker is open source under the MIT license. You can inspect the code on GitHub and verify the installer hash before running.

Why does antivirus flag it?

Some antivirus tools sandbox unsigned Electron apps. This is a false positive. Verify the hash, then add an exception after confirming integrity.

Will it work on Mac or Linux?

Currently only Windows is officially supported. Mac and Linux builds may be available in future releases based on community interest.

How do I update?

Download the latest installer and run it over your existing installation. Your settings and target groups will be preserved.

Where is my data stored?

All data is stored locally on your machine. Your API key is encrypted with AES-256. No data is sent to external servers beyond Torn's API.

How do I report issues?

Open an issue on GitHub Issues with your version number and steps to reproduce the problem.
